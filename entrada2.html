¿Qué es HTML?
HTML (HyperText Markup Language o "Lenguaje de Marcado de Hipertexto") es el código base que se utiliza para estructurar y dar significado al contenido de una página web. No es un lenguaje de programación, sino un lenguaje de marcado, lo que significa que usa etiquetas (tags) para definir elementos como títulos, párrafos, imágenes y enlaces.

¿Para qué sirve HTML?
HTML permite:
* Estructurar el contenido (títulos, párrafos, listas, tablas).
* Insertar multimedia (imágenes, videos, audio).
* Crear enlaces entre páginas.
* Organizar la información para que los navegadores (Chrome, Firefox, etc.) puedan interpretarla correctamente.

Sin HTML, las páginas web serían solo texto plano sin formato ni interactividad.
