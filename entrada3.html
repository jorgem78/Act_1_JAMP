Entrada 3 - ¿Qué es CSS?

CSS significa "Cascading Style Sheets". Este lenguaje permite darle estilo a las páginas web: colores, márgenes, fuentes, diseños, etc.


CSS son las siglas de Cascading Style Sheets (Hojas de Estilo en Cascada). Es un lenguaje de diseño que se utiliza para dar estilo y formato a las páginas web escritas en HTML. Mientras que HTML se encarga de la estructura y el contenido de una página (como títulos, párrafos y enlaces), CSS define cómo se ve ese contenido: colores, fuentes, tamaños, márgenes, diseños y mucho más.

¿Cómo funciona CSS?
CSS funciona mediante reglas de estilo que aplican diseños a los elementos HTML. Estas reglas consisten en:
* Selector: Indica a qué elemento HTML se aplicará el estilo (por ejemplo, h1, p, .clase, #id).
* Propiedad: La característica que se quiere cambiar (como color, font-size, background).
* Valor: La modificación concreta (por ejemplo, red, 20px, #FFFFFF).

¿Por qué usar CSS?
* Separa el contenido del diseño: Permite mantener el HTML limpio y enfocado en la estructura, mientras que el CSS maneja el aspecto visual.
* Ahorra tiempo: Puedes aplicar estilos a múltiples páginas con un solo archivo CSS.
* Más control: Permite diseños complejos, responsivos (adaptables a móviles) y animaciones.

En conclusión CSS es una herramienta fundamental para cualquier desarrollador web. Sin él, las páginas serían simples y sin estilo. Aprender CSS te permitirá crear sitios visualmente atractivos, profesionales y adaptables a cualquier dispositivo.
